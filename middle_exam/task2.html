IntelliJ 단축키 정리 💻

alt 계열

  • alt + enter : 여러 액션을 제안

    • 에러를 고칠 때도 사용할 수 있고, 람다식에서 이 단축키를 누를 경우, 개인 선호도에 따라 더 길거나 짧은 식으로 고칠 수 있다.

  • alt + cmd + B : 구현 탐색

  • alt + F7 : 모든 위치 검색창에 표시

    • 해당 인터페이스가 사용된 모든 위치가 검색창에 표시된다.

  • alt + ↑/↓ : 선택 영역을 확장/축소

  • alt + cmd + L : 서식을 프로젝트 표준으로 맞추기

    • 현재 파일의 서식을 프로젝트 표준에 맞춘다.

  • alt + shift + cmd + L : CodeFormatting Option

    • 서식 옵션을 선택할 수 있다.

command(cmd) 계열

  • cmd + 1 : 프로젝트 창 열기

    • 프로젝트 창을 열고 닫을 수 있으며, 초점(focus)도 프로젝트 창으로 이동하여 키보드로 조작할 수 있다.

  • cmd + E : 최근 열었던 파일 창 + 도구 창

  • cmd + B : 심볼의 선언으로 이동

    • 필드에서 누르면 필드 선언으로 이동하고, 클래스 이름에서 누르면 클래스 파일로 이동한다.

  • cmd + / : 주석 추가

    • 주석이 자동으로 추가된다.

  • cmd + shift + enter : 구문 완성 기능

    • 문법에 맞게 괄호도 넣고, 커서 위치도 알맞게 조절해준다. 짧은 구문도 쉽게 완성할 수 있다.

  • cmd + shift + a : 액션 검색

    • 모든 액션을 검색할 수 있다.

그 외

  • f2 : 다음 오류, 경고, 제안으로 이동

  • esc : 다른 곳으로 가 있는 커서를 코드 화면으로 이동

    • 마우스를 쓰지 않아도 키보드 안에서 바로 타이핑 가능하게 만든다.또한 변경 사항을 적용하지 않고 모든 팝업 닫기에 이용할 수 있다.

  • ctrl X 2 : run anything(무엇이든 실행가능)

  • ctrl + T : 모든 리팩토링 표시

    • 적용 가능한 리팩토리 옵션을 표시한다.

  • shift X 2 : 모든 검색 기능

    • 무엇이든 검색할 수 있다.
